# Huffman_Coding

Huffman coding implementation

# Запуск

В файл text.txt поместить исследуемый текст.

Выполнить:

``` bash
> python BBuT_laborino2_harderino.py
```

В директории `bin` есть уже скомпилированный exe-файл.
